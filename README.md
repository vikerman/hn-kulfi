## Kulfi Hacker News

Hacker News example built using Kulfi - http://github.com/vikerman/kulfi.

Forked from the Svelte version at https://github.com/sveltejs/hn.svelte.dev/

## Quick Start

```sh
npm i
npm start
```

Make edits to code and the browser reloads.
